# Live Donation Bar

An Electron-based donation overlay tool designed for streamers.

## Project Overview

Live Donation Bar is a desktop application developed for content creators and live streamers. It provides a customizable donation progress bar that can be integrated into OBS through Browser Source.

The project was independently designed and developed, covering UI design, API integration, local server implementation, and software packaging.

## Key Features

- YouTube donation integration
- Twitch donation integration
- ECPay payment integration
- OBS Browser Source support
- Custom donation bar skins
- Custom font and color settings
- Real-time donation updates
- Donation goal tracking
- Configuration save/load support

## Tech Stack

- Electron
- Node.js
- JavaScript
- HTML
- CSS
- Express
- ECPay API
- YouTube API
- Twitch API

## Development Highlights

This project was created to solve a practical problem for live streamers by providing an easy-to-use donation tracking system without requiring users to develop their own overlays.

The application supports multiple donation sources and allows streamers to customize the visual appearance according to their branding requirements.
